# data/

Pasta opcional para versionar backups JSON do app.

Use **Aba Exportar → Backup completo (JSON)** dentro do app e mova o arquivo gerado para cá quando quiser fixar um snapshot da semana.

Exemplo de nomenclatura sugerida:

```
backup_2026-W19.json
backup_2026-W20.json
mestre_linhas_tabelas_2026-05-04.json
```
